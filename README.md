# Android TodoList Practice

안드로이드 Todo 리스트 애플리케이션 연습 프로젝트입니다.

## 📱 프로젝트 개요

Kotlin을 사용하여 개발된 안드로이드 Todo 리스트 애플리케이션입니다.
기본적인 할일 관리 기능을 구현하고 있습니다.

## 🛠️ 기술 스택

- **언어**: Kotlin
- **플랫폼**: Android
- **Build Tool**: Gradle

## 💡 주요 기능

- Todo 항목 추가
- Todo 항목 삭제
- Todo 항목 완료 표시

## 🚀 시작하기

### 필수 요구사항

- Android Studio
- JDK 11 이상
- Android SDK

### 설치 및 실행

1. 프로젝트 클론
```bash
git clone https://github.com/Sprites-in-Elias/android-todolist-practice.git
```

2. Android Studio에서 프로젝트 열기

3. 에뮬레이터 또는 실제 기기 연결

4. Run 버튼 클릭 또는 실행
```bash
./gradlew build
./gradlew installDebug
```
